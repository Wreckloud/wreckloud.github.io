JavaScript 介绍

JavaScript是一种运行在 客户端(浏览器) 的编程语言, 能实现人机交互的效果.

JavaScipt 又分为:

  • ECMAScript(JavaScript语言基础) 和 Web APIs
  • Web APIs 其中又包括 DOM(页面文档对象模型) 和 BOM(浏览器对象模型)

书写位置

与CSS类似, 同样也有以下三种书写方式:

行内书写

代码写在标签内部, 在vue框架会使用这种模式.

内部书写

直接写在html文件里, 用<script src="文件路径"></script>标签包裹住.
规范写法是 写在</body>的上方 , 通常将其放在HTML页面底部.
alert(‘你好, js’)页面弹出警告对话框.

外部书写

代码写在以.js结尾的文件里.
通过<script src="文件路径"></script>标签, 引入到html页面中.
外部js标签中间的代码会被忽略.

关于在JavaScript的结束符, 对于浏览器可以自动推断语句结束位置, 写不写无所谓.

JavaScript 值

JavaScript 语句定义两种类型的值:

混合值被称为字面量（literal）.
变量值被称为变量.

有些数据可以在程序使用之前预先设定并在整个运行过程中没有变化，这称为常量。

有些数据在程序运行中可以变化或者被赋值，这称为变量。

变量命名规整

规则:

  • 不能使用关键字
  • 只能使用 下划线, 字母, 数字, $ 组成, 不能以数字开头
  • 字母严格区分大小写

规范:

  • 起名有意义
  • 遵守小驼峰命名法

变量的基本使用

要想使用变量, 首先需要创建变量(也称为声明变量或者定义变量).

使用 var 重复声明不会丢失它的值.

使用 let 关键字重新声明变量可以解决这个问题

常量

使用 const 声明的变量称为 “常量” .
当某一个变量永远不会被改变的时候, 就可以使用 const 来声明.

常量不允许重新赋值, 声明时必须初始化.

数据类型

number 数字型

JS属于弱数据类型, 数字类型(Number)可以是整数, 小数, 正数, 负数.

NaN(Not a number, 非数) 表示一个计算错误. 对NaN的任何操作都会返回 NaN.

boolean 布尔型

只有两个值: true 和 false

String 字符串型

被 “”(双引号) ‘’(单引号) ``(反引号) 包裹的都是字符串.

+ 运算符, 可实现字符串拼接.

数组

JavaScript 数组用方括号书写.
数组属于对象.

引用数据类型

object 对象

对象用花括号来书写

undefined 未定义型

只声明变量而 不赋值, 使用未定义型.

null 空类型

Javascript 中的 null 仅仅是一个代表 “无”, 或 “值未知” 的特殊值.
Undefined 与 null 的值相等，但类型不相等.

空值

空值与 undefined 不是一回事.

空的字符串变量既有值也有类型.

正则表达式

正则表达式可用于执行所有类型的文本搜索和文本替换操作.

修饰符

正则匹配到第一个匹配项则停止, 并且默认大小写敏感, 如果想修改默认选项, 则需要修饰符.

修饰符 描述
i:ignore case 执行对大小写不敏感的匹配
g:global 执行全文搜索 (查找所有匹配而非在找到第一个匹配后停止)
m:multiple lines 执行多行匹配

正则表达式

  1. 括号用于查找一定范围的字符串
    表达式 描述
    [abc] 查找方括号之间的任何字符
    [0-9] 查找任何从 0 至 9 的数字
    (x y)
  2. 元字符（Metacharacter）是拥有特殊含义的字符
    元字符 描述
    \d 查找数字
    \s 查找空白字符
    \b 匹配单词边界
    \uxxxx
  3. Quantifiers 定义量词
