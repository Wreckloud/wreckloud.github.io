C# 简介

C# 是 .Net(.Net Framewor) 框架的一部分，且用于编写 .Net 应用程序.
与Java类似, C# 也是一种面向对象的编程语言.

C#中, 程序会从: Main 函数 开始执行, 以 ; 分号结束语句, 一行可以有多条语句.

与 Java 不同的是，文件名可以不同于类的名称。

数据类型

常量

常量用关键字 const 修饰. 不能被修改.

变量

在 C# 中，变量分为以下几种类型:

  • 值类型（Value types）
  • 引用类型（Reference types）
  • 指针类型（Pointer types）

值类型

值类型直接包含数据. 其常用值类型有:

类型 说明
int 有符号整型, 可以是负数. (uint无符号整型)
float 精度大约6-9位数字
double 精度大约15-17位数字
bool 取值有true或false
char 存储单个字符

使用 sizeof() 方法, 可以获取指定类型的存储大小.

引用类型

引用类型指的是一个内存位置.

类型 定义语法 说明
字符串（String）类型 1. String str = "这是一个字符串";
2. @"这是一个字符串";		 字符串是一种特殊的对象,由多个char组成.第二种称为逐字字符串,它不可以使用转义字符.
对象（Object）类型 1.装箱(值类型转换为引用类型) :
int i = 123; object box = i;
2.拆箱(引用类型转换为值类型)
int i = (int)box;		 对象类型可以被分配任何其他类型,
装箱和拆箱能让引用类型和值类型相互转换
(但会带来性能上的损失)
动态（Dynamic）类型 dynamic 变量名 = 值; 可以绕过静态类型检查

指针类型

指针类型变量存储另一种类型的内存地址, C# 中的指针与 C 或 C++ 中的指针有相同的功能.

值类型* 指针名字;

要执行的语句; 至少会被执行一次.

for 循环

初始化值 会首先被执行, 且只会执行一次.
接下来会判断 条件 是否为真, 若为真就执行 要执行的语句; .反之跳出 for循环.
执行完主体之后, 执行 增量 更新循环控制的变量.

foreach 循环

C# 也支持类似于Python中for循环的 foreach循环, 使用 foreach 可以挨个操作数组或者一个集合对象.
