C# 简介

C# 是 .Net(.Net Framewor) 框架的一部分，且用于编写 .Net 应用程序.
与Java类似, C# 也是一种面向对象的编程语言.

C#中, 程序会从: Main 函数 开始执行, 以 ; 分号结束语句, 一行可以有多条语句.

C# 文件的后缀为 .cs

Console.WriteLine() 能在控制台输出内容. 相对的, 控制台输入函数是: ReadLine()

Console.ReadLine() 只接受字符串格式的数据.

变量类型 说明
int 有符号整型, 可以是负数
uint 无符号整型, 不能表示负数
float 精度大约6-9位数字
double 精度大约15-17位数字
bool 取值有true或false
char 存储单个字符
string 存储字符串, 由多个char组成.
数组 数组类型[] 数组名 = new 数组类型[数组长度]

使用 sizeof() 方法, 可以获取指定类型的存储大小.
