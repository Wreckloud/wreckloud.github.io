MySQL 概述

数据库

数据有组织地存储的仓库, 称为 数据库 (DataBase).
数据库管理系统(DataBase Management System) 是能操作和管理数据库的大型软件. MySQL 就是一个数据库管理系统.

SQL (Structured Query Language) 是操作关系型数据库的编程语言, 定义了一套操作关系型数据库统一标准.
它不仅仅能操作 MySQL, 其他主流的 关系型数据库管理系统 (如:ORACLE, SQL Server…) 也都统一标准操作.

MySQL 数据库

关系型数据库 (RDBMS)

建立在关系模型基础上, 由多张相互连接的二维表组成的数据库.

在一个数据库服务器中可以创建多个 数据库 .
在一个 数据库 中 又可以创建多个 表 .
基于 表 进行数据存储的数据库, 我们称之为 关系型数据库.

SQL 通用语法

SQL语句以分号结尾, 空格缩进和换行都不影响, 也不区分大小写, 不过还是推荐关键字使用大写.

注释

SQL 分类

类别 说明
DDL (Data Definition Language) 数据库定义语言, 用来定义数据库对象(数据库, 表, 字段)
DML (Data Manipulation Language) 数据库操作语言, 用来对数据库表中的数据增删改查
DQL (Data Query Language) 数据查询语言, 用来查询数据库中表的记录
DCL (Data Control Languag) 数据控制语言, 用来创建数据库用户, 控制数据库的访问权限

MySQL 常用数据类型

主要分为三类: 数值类型, 字符串类型, 日期时间类型.

数值类型

类型 描述
Binary 二进制串, 固定长度 n.
Boolean 存储 TRUE 或 FALSE 值
Int 或 IntEger 长整数, 4字节
SmallInt 短整数, 2字节
Float 单精度浮点数值
Double 双精度浮点数值
Decimal -

字符串类型

类型 描述
Char(n) 定长字符串, 固定长度 n.
VarChar(n) 变长字符串, 最大长度 n.

日期

类型 描述
DATE 存储年、月、日的值 (YYYY-MM-DD)
TIME 存储小时、分、秒的值 (HH:MM:SS)
TIMESTAMP 存储年、月、日、小时、分、秒的值
INTERVAL 由一些整数字段组成, 代表一段时间, 取决于区间的类型
