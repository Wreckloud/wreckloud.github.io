初识 Unity

基础界面简介

项目文件结构

在新建完成后会在创建的目录下生成相关工程文件:

Assets: 资源（场景脚本模型）
Library: 库（系统）
Logs: 日志
Packages: 导入的包
ProjectSettings: 工程设置
Temp: 临时文件（文件过大可以删除部分缓存）
UserSettings: 设置

Unity 窗口

接下来打开 Unity 窗口, 会有5个窗口面板:

Project

工程面板，存放工程的各种资源。包括：声音资源、模型资源、场景文件、材质、脚本等。

创建代码

在 Project(工程面板) > Assets 右键选择 create 创建 C# script ,
并重命名为 New Name 右键即可打开

Unity支持持的编程语: C#、Mono、JavaScript

Hierarchy

层级面板，展示当前打开的场景里面有哪些东西。（里面的东西称为游戏物体）

Inspector

检视面板（或属性面板），查看一个游戏物体由哪些组件组成。
场景 -> 多个游戏物体 -> 多个组件

Scene

场景面板，显示当前场景的样子

场景基本操作

聚焦：双击游戏物体 或者 F
放大缩小视野：鼠标滚轮
围绕物体旋转：Alt+鼠标左键

工具栏

Q：视图工具（使用鼠标中键可以临时切换到这个工具）
W：移动工具
E：旋转工具
R：缩放工具

Game

游戏面板，场景运行的时候的样子
