更新 Nginx Proxy Manager

万事先备份, 以防万一

不要使用 docker-compose stop 来停止容器, 因为这么做需要额外的时间等待容器停止.

使用 docker-compose up -d 直接升级容器时会自动停止并立刻重建新的容器, 完全没有必要浪费那些时间.

prune 命令用来删除不再使用的 docker 对象。删除所有未被 tag 标记和未被容器使用的镜像
