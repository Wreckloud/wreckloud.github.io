VUE 概述

Vue 是一套 渐进式 前端框架 , 可以按需引入Vue.js的部分功能.

VUE官网

传统网页的内容都是写死了的. 而 VUE 所作的, 就是用一个变量或者占位符代替, 将其变成动态内容页面.

看完这个视频, 我就不信你还学不会Vue

VUE 快速开始

传统方式开发

可以借助 script 标签直接通过 CDN 引入 Vue.js 文件(也可以下载这个文件 vue.global.js ).

获取简单数据类型

  1. 在JS代码区域, 创建Vue核心对象, 定义数据模型.
  1. 在页面上创建 id 为 app 的元素.
  1. 将 VUE 应用程序 挂载到页面 app 元素上, 接收返回值

获取复杂数据类型

当需要获取对象,数组之类的, 就需要使用到 reactive.

reactive 是一种响应式数据的类型, 先放一边, 待会会简单解释一下.

使用 = 即可修改 reactive 对象的属性.

可以使用 解构 的方式让代码简洁一丢丢.

响应式数据

创建响应式数据是指建立一种机制, 使得当数据发生变化时, 视图(用户界面)能够自动更新以反映这些变化.
这是Vue的核心特性之一, 它允许开发者以声明式的方式编写代码, 而无需手动操作DOM.

响应式数据的类型先介绍这两个:

  • reactive 用于存储复杂数据类型的响应式对象.
  • ref 用于存储单个基本类型的响应式数据.

使用 .Value 修改 ref 中的值

ref 也可用于存储数组数据.

模块化开发

只需要对传统方式开发稍作修改即可转变为模块化开发.

但需要注意的是, 本地路径无法显示模块化开发. 使用 VScode 下载插件 Live Server 以使用本地服务器调试即可.

需要引入 vue.esm-browser.js , 同样可以事先下载然后本地引入.

只需在 script 标签添加 type="module" 以及引入 vue.esm-browser.js 即可完成从传统模式到模块化的转变.

VUE 事件

v-model="值"|在表单元素上创建双向数据绑定

v-if
v-else
v-else|条件性渲染元素, 判定为true时渲染, 否则不渲染

v-on 绑定事件

v-on:事件="事件名()" 能为HTML标签绑定事件, 例如绑定点击事件 click 使每次点击按钮, 数值加一:

click 就是一个标准的DOM事件, 除此之外常用的事件类型有:

鼠标事件

事件名称 触发时机
@mouseover
@mouseout		 鼠标指针 移入\移出 到元素上时
@mouseenter
@mouseleave		 鼠标指针 移入\移出 元素边界时
@dblclick 双击元素时
@mousedown
@mouseup		 鼠标按钮 被按下\被释放 时
@mousemove 鼠标指针在元素上移动时

键盘事件

事件名称 触发时机
@keydown 键盘按键被按下时
@keyup 键盘按键被释放时
@keypress 键盘按键被按下并保持一段时间时

当使用键盘事件, 一般会监听特定的按键.
例如, 监听键盘上的Enter键: @keyup.enter.

此时, 只有在松开键盘上的Enter键时, 才会触发keyup事件.

同样的,
如果需要监听松开 空格, 就写 @keyup.space ,
如果需要监听按下 Tab, 就写 @keydown.tab ,
当需要组合按下CTRL+A时, 就写 @keydown.ctrl.a
依次类推.

v-show 显示控制

在元素中添加 v-show:布尔值 属性可以实现显示控制.

当 v-show:false 时, 会给元素添加 style="display: none;" 的 CSS 属性, 以达到隐藏元素的效果.
当 v-show:true 时, 会移除该属性.

可以配合 绑定事件 以达到切换状态的效果

V-if 条件渲染

v-if:布尔值 也可以实现显示控制, 不过它是删除/添加 DOM 元素.

当 v-show:false 时, 会移除该元素, 以达到隐藏元素的效果.
当 v-show:true 时, 会渲染这个元素.

v-if:布尔值 不适合频繁切换显示隐藏的元素.

v-if 的使用方法与其他编程语言中的条件语句类似.
例如, 需要按条件显示状态.

v-for 列表渲染

v-for 能遍历容器的元素或者对象的属性.

这样, v-for 就能将数组中的元素遍历出来, 并用 li 标签依次展示出来.

如果遍历的是对象, 直接使用 v-for 会将所有属性依次展示出来.
但一般我们都希望属性以 key:value 的形式, 可以这样写.

v-bind 动态属性绑定

v-bind:属性值 能为HTML标签绑定属性值, 如设置 href, css 样式等.
v-bind: 可简写为 :.

例如, 动态绑定文本输入框的提示文本或动态绑定超链接.

v-model 双向数据绑定

不同于 v-bind 动态属性绑定, 双向数据绑定能使用户修改input值时, 自动更新数据.

v-model 是实时渲染的,
使用 .lazy 能使其在失去焦点或按下回车之后再渲染.
使用 .number 能使输入框的值转换为数字.
使用 .trim 能去除字符串首尾的空格.

v-text/v-html 渲染数据

v-text="字符串" 会将字符串解析为纯文本格式.
v-html="字符串" 会将字符串中的HTML标签解析为HTML格式.

computed 计算属性

普通的函数不会自动追踪依赖, 每次调用都需要手动传入参数.
computed 属性是响应式的, 它会根据依赖的数据自动重新计算.

computed 属性具有缓存机制, 只有当依赖的数据发生变化时, 计算才会重新执行.
这意味着在依赖数据未变化的情况下, 多次访问同一个computed属性会立即返回之前的计算结果, 而不会重复执行计算.
普通的函数每次调用都会执行.
